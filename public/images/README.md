# App Images

Place app images in this directory:

- `app-icon.png` - App icon for notifications
- `badge-icon.png` - Badge icon for notifications  
- `default-avatar.png` - Default user avatar
- `favicon-16x16.png` - 16x16 favicon
- `apple-touch-icon.png` - Apple touch icon

All images should be in PNG format with appropriate sizes:
- Icons: 192x192 or 512x512
- Avatars: 80x80 or 120x120
- Favicons: 16x16, 32x32
